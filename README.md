# Churreros App
Aplicación web simulando compra y venta de churros.

## Funcionalidades
- Búsqueda de churreros
- Creación y cancelación de pedidos
- Envío de mensajes
- Administración de productos

## Tecnologías
- HTML
- CSS
- JavaScript
- LocalStorage

## Testing (QA)
Casos probados:
- Creación de pedido sin productos → error controlado
- Mensaje vacío → validación
- Actualización de precio inválido → bloqueado
- Búsqueda sin resultados → feedback al usuario

## Mejoras futuras
- Login de usuario
- Backend con API
- Automatización de tests
