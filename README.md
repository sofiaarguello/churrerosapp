# Churreros App

Aplicación web que simula la compra y venta de churros, permitiendo la interacción entre usuarios y vendedores.

## 🚀 Funcionalidades
- Búsqueda de churreros
- Creación y cancelación de pedidos
- Envío de mensajes
- Administración de productos

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript
- LocalStorage

## 🧪 Testing (QA)
Se validaron distintos escenarios para asegurar el correcto funcionamiento:

- Creación de pedido sin productos → error controlado
- Envío de mensaje vacío → validación implementada
- Actualización de precio inválido → bloqueado
- Búsqueda sin resultados → feedback al usuario
